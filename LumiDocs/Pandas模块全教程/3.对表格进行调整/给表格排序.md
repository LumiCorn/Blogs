# 给表格排序

在遇到有关成绩或者分数的需要排名的表格，我们一般会对总分进行排序。目前我们有两种不同的排序方法：升序和降序。
- **升序** 是指从最小的数字开始，越往下数字越大。
- **降序** 是指从最大的数字开始，越往下数字越小。

在对df表格进行排序操作时，我们只需要定义一个描述标签：`ascending=`。

先让我们导入一个表格吧！
```python
import pandas as pd

score_sheet = [
  [“Lumi”,89]
  [“Jisoo”,92]
  [“JungKook”,97]
]

df = pd.DataFrame(score_sheet,columns=“Name”,”Score”)

print(df)
```
